# E-Project-Semi4

1. Clone the Project
2. 
```bash
# install npm
> npm install
```
3. Create .env file
```bash
# copy this to .env file
VITE_BASE_API_URL=https://hot-spring-raclette.glitch.me/
```
4. Run project
```bash
# npm
> npm run dev
```

```bash

  ➜  Network: http://192.168.137.1:5173/
```
